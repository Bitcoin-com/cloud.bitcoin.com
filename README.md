# Cloud.bitcoin.com

Web GUI for Bitcoin.com's Cloud

## Usage

### Installation

Clone the repo

`git clone https://github.com/Bitcoin-com/cloud.bitcoin.com.git`

Install dependencies

`cd cloud.bitcoin.com && yarn install`

Start app

`yarn develop`

Open your browser to `http://localhost:8000/` to see the app.
